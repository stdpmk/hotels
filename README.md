# Сервис бронирования отелей

Упрощённый REST API-сервис для поиска и бронирования отелей.

---

## Основной функционал

### Отели
- Получить список отелей с фильтрацией по городу, цене, рейтингу
- Получить детальную информацию об отеле (название, адрес, описание, рейтинг, фото)
- Поиск доступных отелей по датам заезда/выезда и количеству гостей

### Номера
- Список номеров отеля с указанием типа (одноместный, двухместный, люкс и т.д.)
- Детали номера:
  - Цена за ночь
  - Максимальное количество гостей
  - Количество комнат и кроватей
  - Допустимые условия: дети, животные, курение
  - Удобства: Wi-Fi, кондиционер, завтрак включён и т.д.
- Проверка доступности номера на выбранные даты

### Бронирование
- Создать бронь (номер + даты + данные гостя)
- Автоматический расчёт итоговой стоимости (цена за ночь × количество ночей)
- Просмотр деталей брони по номеру бронирования
- Отмена брони

### Пользователи
- Регистрация и авторизация
- Просмотр истории своих бронирований
- Профиль пользователя

---

## Стек технологий

- **Язык:** Go
- **База данных:** PostgreSQL
- **Роутер:** Gorilla Mux
- **Аутентификация:** Bearer token
- **Документация:** OpenAPI
- **Кеши очереди:** redis, kafka
